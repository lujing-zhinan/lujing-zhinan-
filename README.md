<div class="card">
  <h2>② 你能报初级还是中级？（自查表）</h2>
  <p><span class="tag">初级（五级）</span>满足任一条即可：</p>
  <ul>
    <li>年满16周岁 + 拟从事/已从事消防中控/物业安全员相关工作</li>
    <li>已从事相关工作满 <strong>1年</strong>（单位证明能佐证）</li>
  </ul>
  <p style="margin-top:.8rem"><span class="tag">中级（四级）</span>满足任一条即可：</p>
  <ul>
    <li>已有初级证 + 再干满 <strong>4年</strong></li>
    <li>没初级证，但累计干满 <strong>6年</strong>（最常见路径）</li>
    <li>技校/中职相关专业毕业（含应届）</li>
  </ul>
  <p style="margin-top:.6rem;color:#555">📌 学历硬门槛：<strong>高中或同等学力及以上</strong>（中专/职高/技校都算）</p>
</div>

<div class="card">
  <h2>③ 自己报名七步走（官网流程直译）</h2>
  <ol>
    <li><strong>注册账号</strong>：进 xfhyjd.119.gov.cn → 考生注册（手机号收验证码）</li>
    <li><strong>完善资料</strong>：身份证号/学历/工作经历（经历要与工作证明文字对上）</li>
    <li><strong>上传附件</strong>：身份证正反面、学历证、工作证明、证件照（注意白底/蓝底规格）</li>
    <li><strong>等初审</strong>：状态变绿=过；被驳回应看驳回原因逐条改（最常见：工作证明信息对不上/照片规格错）</li>
    <li><strong>缴费</strong>：初审通过后才出现缴费入口（短信也可能发缴费码）</li>
    <li><strong>选场次</strong>：收到预选短信后尽快登回系统自选时间（好时段秒没）</li>
    <li><strong>打准考证 → 考试日带齐证件</strong></li>
  </ol>
</div>
